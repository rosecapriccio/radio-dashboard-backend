ラジオ実況ツイートダッシュボード

機能  
・ラジオ番組の公式ハッシュタグでつぶやかれたツイート文章／ツイート時間をPythonで収集、DBに保存  
　→番組名と放送日ごとに  
 ・ツイート数の推移グラフ：　放送時間（２時間）のツイート数の推移をグラフ表示。番組の盛り上がり箇所／ピークスポットが一目でわかる  
 ・ワードクラウド機能：　ツイート文章を形態素解析＋名詞のみ抽出し、ワードクラウドとして表示。放送日の番組の大まかな内容、トピック、パンチラインを把握できる
