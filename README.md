# 📻 ラジオ実況ツイートダッシュボード

（X API仕様変更に伴い改修中）  
ラジオ番組の放送中に公式ハッシュタグで投稿された膨大なリアルタイム実況ツイートを収集・分析し、番組の「盛り上がり」や「トレンドトピック」を視覚的に捉えることができるデータ分析ダッシュボードアプリです。

## 🚀 主な機能

### 1. リアルタイム・ツイートデータ自動収集（Python）
- X（Twitter）APIを活用し、指定した番組の公式ハッシュタグを含むツイート本文および投稿時間を自動収集。
- 番組名（識別ID）と放送日ごとに分け、リレーショナルデータベースへ格納。

### 2. 時系列ツイート量推移グラフ（React）
- 2時間の放送時間内におけるツイート投稿数の増減を時系列グラフ（ラインチャート）で表示。
- どのコーナー、どの発言のタイミングでタイムラインが最も盛り上がったか（ピークスポット）を一目で可視化。

### 3. 自然言語処理を用いたワードクラウド生成（React）
- 収集したツイート文章を形態素解析エンジンによって分解し、名詞のみを抽出。
- 放送日に多く呟かれたバズワードやパンチライン、番組トピックをテキストマイニングの技術（ワードクラウド）でグラフィカルに表現。
